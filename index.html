Canon Protocol

A specification for structured, versioned, and verifiable data transformations.

What is Canon Protocol?

Canon Protocol defines a standard for creating, versioning, and distributing specifications that transform unstructured data into structured formats. It uses cryptographic signatures for verification and DNS-based identity for publishers.

Quick Start

cargo install canon-cli canon init canon install

See the documentation for detailed instructions.

Core Specifications

  • canon-protocol.org/type@1.0.0 Defines how to create new specification types
  • canon-protocol.org/transformation@1.0.0 Defines transformations from source data to structured output
  • canon-protocol.org/auto-version@1.0.0 Automatically determines semantic version changes

Publisher Identity

This domain serves as the publisher identity for canon-protocol.org specifications. Public keys for signature verification are available at:

https://canon-protocol.org/.well-known/canon-keys.json
